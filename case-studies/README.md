# AI Model Evaluation Case Study

This folder contains my first independent case study, where I simulate an AI model evaluation workflow using a 300-record dataset.

- Focus: model accuracy, safety prompts, hallucinations, bias, and human-in-the-loop tradeoffs.
- Tools: Google Sheets (pivot tables, formulas, charts).
- Key finding: at maximum confidence, the model achieves only about 50% accuracy, revealing “hallucinations with certainty.”

The PDF in this folder is the full writeup with charts and links to the underlying spreadsheet.
